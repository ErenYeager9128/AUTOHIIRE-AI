### Getting Started
- Clone the repository:  
  `git clone https://github.com/autohire-ai.git`
- Change directory:  
  `cd autohire-ai`
- Create a virtual environment:  
  `python -m venv venv`
- Activate it:  
  `source venv/bin/activate`  # On Windows: `venv\Scripts\activate`
- Install dependencies:  
  `pip install -r requirements.txt`
